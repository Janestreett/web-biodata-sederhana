## 📂 Daftar Tugas

### 1. TUGAS 1: Pengenalan Dunia Website
* **Deskripsi:** Berisi penjelasan teori dasar mengenai definisi website, fundamental HTML, fungsi utama web browser, serta berbagai perangkat (*tools*) yang digunakan oleh pengembang web.
* **Format:** Dokumen / Teks (Minimal 300 kata).

### 2. Praktik 1: Website Pertama
* **Deskripsi:** Praktik pertama membuat file `index.html` sederhana untuk menampilkan judul, nama, kelas, dan paragraf biodata dasar.
* **Teknologi:** HTML5.

### 3. TUGAS 2: Website Biodata Sederhana
* **Deskripsi:** Pengembangan dari praktik pertama dengan menerapkan struktur HTML yang lebih rapi menggunakan elemen tag `heading`, `paragraf`, dan `line break (<br>)`. Di sini saya menceritakan ketertarikan saya di bidang finansial dan pasar saham.
* **File:** `biodata.html` / `index.html`
